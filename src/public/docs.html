Url Shortener API

Usage

Create a new URL short-link

    
      {
        "method": "POST",
        "path": "/api",

        // What you are sending
        "body": {
          "path": "https://www.google.com"
        },

        // All is ok (200)
        "payload": {
          "path": "https://www.google.com",
          "pureId": "1492432",
          "encodedId": "gqpE"
        }
      }

Get a shortened-link

    
      {
        "method": "GET",
        "path": "/api/gqpE",

        // What you are receiving
        "response": "https://www.google.com"
      }
